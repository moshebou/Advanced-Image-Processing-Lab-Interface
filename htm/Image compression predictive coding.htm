3         Image compression: predictive coding

Image compression is an important field in the area of image processing.

Image compression method can be grouped into three steps:

decorrelating image sampled representation;

scalar quantization of decorrelated data;

binary statistical encoding of quantized decorrelated data;

Decorrelating image sampled representation is a processing that converts set of image or video samples into a set of discrete data that are statistically, i.e. regarded as a statistical ensemble, uncorrelated as much as possible. This is achieved by applying to them one or another decorrelating linear transformation from two groups of transformation: predictive transformations and orthogonal transforms.

3.1         Prediction

3.1.1     Theoretical background

Predictive transformation compute pixel-wise differences between pixel gray level and its predicted estimate found as a weighted sum over pixels that surround the given one on a sampling grid. These differences are then subjected to scalar quantization. The method, in which the prediction and computing differences is performed in course of image row-wise/ column-wise scanning, is one of the most earliest method of image and signal compression. It is called Differential Pulse Code Modulation (DPCM). Nowadays it is used mostly for “inter-frame” coding of video sequences.

In DPCM, the prediction of pixel values is made over only immediate vicinity of pixels preceding the given one in the process of row-wise scanning [1].

The “Prediction” experiment gives example of 3 different DPCM prediction for image pixels– 1D horizontal, 1D vertical, and 4-neighbor 2D prediction. 3.2.2     Experiment Description

The “DPCM” experiment provides a simulation of DPCM encoding – quantize – transmission – decoding scheme with 1D and 2D predictions, with additive Gaussian noise or impulse noise.

In both cases the DPCM scheme is similar:

Calculate the difference between current sample and the prediction of the correct sample using reconstructed samples, as described in ‎3.2.1 if the difference is bigger the threshold, use the threshold instead. 

Quantize the difference using P-law quantization.

Add noise (additive Gaussian or impulse noise, defined by the user)

Restore the current result sample using inverse quantization of the noisy quantized difference.
