2.1         Vision Contrast Sensitivity

2.1.1     Theoretical background

A natural primary requirement for quantization of image gray levels is to secure invisibility of false contours in displayed digital images. The threshold of visibility of patches of constant gray level depends on their contrast with respect to their background and of size of patches. Quantitative estimates can be made using Weber-Fechner’s law illustrated in Figure ‎2‑1 . 

Figure ‎2‑1: The Weber-Fechner law on the threshold visibility of a stimulus on a uniform background

2.1.2     Experiment description

The “Vision Contrast Sensitivity” illustrated the Weber-Fechner’s law using an interactive experiment.

The user required to detect the stimulus on a uniform background, using the mouse courser. If the user misses the stimulus, it relocates to a new random location and the contrast increses in one. If the user succeeds in locating the stimulus, the stimulus size increases by two pixels in all axes. After 7 successful detections of stimulus location, the background changes to the next one.

All values are in 8 bits. Contrast is in the range of 1-35, the backgrounds are [16, 128, 220], and stimulus size is 1,3,5,7,9,11,13,15.

2.2.1.3     Compressor-expander (compander) approach

Compressor-expander quantization was suggested as a method for hardware implementation of non-uniform optimal quantization using readily available uniform quantizers in which signal dynamic range is split into quantization intervals of equal size, and centers of the intervals are used as quantization levels. In order to implement a non-uniform quantization with uniform quantizers, signal, before being sent to the uniform quantizer is subjected to a nonlinear point wise transformation. Correspondingly, at the signal reconstruction stage, uniformly quantized values have to be subjected to a nonlinear transformation inverse to the one used for quantization. Usually, the required non-linear pre-quantization transformation compresses the signal dynamic range, thus the name “compressor-expander quantization”. 2.2.3     Results

Figure ‎2‑6: “Signal quantization”. Top left is the test signal. Top right is a quantized version of the signal, using max Lloyd quantization. Middle left plot is a quantized version of the signal, using uniform quantization. Middle right plot is a quantized version of the signal using p-law quantization. Bottom plot is the signal histogram, and the luts used for quantization. Test signal was originally 8-bit, and all quantization methods used 7-bit quantization.

Figure ‎2‑6 shows that for the given signal, Lloyd max quantization is the best quantization method in the RMS sense. The worst is the uniform quantization. This result is very intuitive – the Lloyd max quantization was designed to minimize the RMS error, and it is very signal adaptive, i.e. it take into consideration the signal characteristics (it’s psf), while the uniform quantization considers only the minimum and maximum values of the signal (and for some implementation, not even that), which makes it suitable for uniformly distributed signals.

The P-law quantization (for P smaller then 1) allocates more quantization levels to small amplitude value (can be seen at lut of the P-law quantization at Figure ‎2‑6, notice that the signal have both positive and negative values), and it is more suitable for signals with higher probability to smaller absolute value. The P-law quantization can be further optimized by calculating for each input signal its optimal P value.

Although the Max-Lloyd seems to give the optimal result, it is not guaranteed, for two main reasons.

The first reason is a technical reason. If the signal is corrupted by noise, and then quantized, the Max-Lloyd quantization will produce an optimal quantization for the corrupted signal, and not for the original signal, which result in high RMS error.

The second reason is the face that Max Lloyd quantization does not guarantees conversion to the global minimum, only to the local minimum.

Consider the following example illustrated at Figure ‎2‑7:

2.4         Image Spectral Quantization

2.4.1     Theoretical background

The P-th law compression transformation, thanks to its simple parameterization through a single parameter proved to be a useful approximation to optimal compression transformation, in particular, for quantizing Discrete Fourier Transform (DFT) and Discrete Cosine Transform (DCT) coefficients of images in transform image coding. Optimal values of the nonlinearity index P for which standard deviation of errors in reconstructed images because of quantization of their spectra is minimal are usually about 0.2-0.3. [2]
